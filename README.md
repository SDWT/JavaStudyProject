## Module 4:
1) Реализовать программы, которые получают DeadLock и LiveLock.
2) Создать два потока (поток 1 постоянно выводит "1", а поток 2 выводит "2"), которые будут бесконечно выводить в консоль "1" и "2" по очереди(начиная с "1")

###Реализация
1.1) DeadLock был основан на последовательной блокировки 2 объектов.
- Первый поток блокирует сначала первый объект, затем пытается заблокировать второй, не отпуская первый.
- Второй поток делает тоже самое, но меняет объекты блокировки местами сначала второй, затем первый.
1.2) В случае с LiveLock, при неудачи блокировки второго объекта, он отпускает первый объект.
2) Вывод последовательно "1 2 1 2 ..." выполняется по средствам массива conditons, где поток вызывает следующий поток для вывода следующего значения.
