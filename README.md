# JavaStudyProject
Homework Java course.

## Module 1:
Реализовать свой иммутабельный класс, который будет внутри себя содержать поле с изменяемым классом.

Реализация
Car - изменяем класс;
CarHolder - иммутабельный класс, имеет свойства в виде даты создания и списка объектов типа Car;
Main содержит проверку неизменности внутренностей экземпляра класса CarHolder.
